-  npm i baoth-fast-cache
-  CDN http://unpkg.com/baoth-fast-cache/release/bundle.js

## 快速使用
```js
var FC = window.FastCache;
var cache = new FC();
cache.set('a', 100);
alert(cache.get('a'));
```
